Mục Tiêu Nghề Nghiệp

Vận dụng các kiến thức về thuật toán, kiến trúc phần mềm và kỹ năng lập trình (Java, C#, Android) để xây dựng các ứng dụng có hiệu suất cao. Luôn sẵn sàng học hỏi công nghệ mới và giải quyết các bài toán kỹ thuật phức tạp.

Kỹ Năng Kỹ Thuật

  • Ngôn ngữ & Framework: Java (JSP, Servlets, JDBC), C# (.NET), Android SDK.
  • Cơ sở dữ liệu: MySQL, SQL Server.
  • Kỹ năng khác: Lập trình hướng đối tượng (OOP), Mô hình MVC, Đọc/ghi dữ liệu JSON, Giải thuật (ICPC).

Dự Án Tiêu Biểu

  • Hệ thống Quản lý Sản phẩm (Web App)
    Công nghệ: Java MVC, Tomcat, MySQL
    Xây dựng hệ thống phân quyền và thực hiện các thao tác CRUD để quản lý hàng hoá trong kho.

  • Ứng dụng Danh bạ Sinh viên (Android)
    Công nghệ: Android Studio, Java, JSON
    Phát triển ứng dụng di động đọc dữ liệu từ file assets JSON, hiển thị danh sách qua ListView tuỳ chỉnh và xử lý lưu trữ bộ nhớ.

  • Phần mềm Quản lý Nhân sự
    Công nghệ: C# .NET, OOP
    Thiết kế kiến trúc phần mềm dựa trên biểu đồ UML, áp dụng tính kế thừa và lớp trừu tượng (abstract classes).
